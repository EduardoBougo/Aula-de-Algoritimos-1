# Aula-de-Algoritimos-1
Repositorio voltado a armazenagem de arquivos utilizados ou criados em sala de aula.

Passo a passo para usar o repositorio no computador estudantil:
    - Logar no GitHubDesktop
    - Clonar o repositorio "Aula-de-algoritimos"
    
    - Trocar a linguagem padrão do VsCode de "UTF-8" para "ISO 8859-1"
        + Clique na engrenagem (botão de gerenciar) no lado esquerdo do VSCode
        + Selecione a opção "Configurações"
        + Digite "encoding"

    - Abrir a pasta do repositorio no VsCode
    - Baixar as extenções "C/C++" e "C/C++ Compile Run"
